## Общие паттерны успешных (must-have)
1. Бесплатный вход обязателен (диагностика / пробный / лид-магнит)
2. Личный бренд педагога в hero (фото)
3. Цифры учеников/лет на видном месте
4. Видеоотзывы > текстовых
5. Мессенджеры (TG + WA) как основной канал связи
6. Концерты/выступления учеников как доказательство результата
7. Tilda — де-факто стандарт (быстро, адаптив)

## Каналы рекламы (бенчмарки 2026)
- **VK-таргет:** лид 170–520 ₽. Формат-победитель — видео из реальных занятий (телефонное > студийного). ЦА: женщины 21–47.
- **Яндекс.Директ:** лид 800–1000 ₽. Горячие запросы «уроки вокала + город». После оптимизации CR сайта до 11%.
- **Органика/YouTube:** главный двигатель goloslogos (380k подписчиков как донор).
- **Медиа-микс 2026:** Директ + VK/ОК + Telegram + блогеры + YouTube.

## Что забрать на наш сайт (приоритизировано)

### P0 — критично для конверсии платного трафика
1. **Цифры в hero / под ним** — сейчас ноль. Минимум: учеников, специалистов, лет практики.
2. **Видео** — атмосфера школы / видеоотзывы. Сейчас только фото.
3. **Лид-магнит для «не готовых»** — чек-лист / аудио-упражнение / тест голоса за email. Догрев тех, кто не запишется сразу.

### P1 — доверие и качество лидов
4. **Команда с реальными фото + регалии** (степень, где учились, проекты).
5. **Гарантия** результата/возврата.
6. **Выступления/записи учеников** — «куда приводит программа».

### P2 — рост конверсии
7. **Мессенджер-виджеты** (TG + WA), плавающие.
8. **Блок «что именно на диагностике»** — снять страх неизвестности (✓ уже частично есть — getafter + process).
9. **Живые отзывы** — имена + фото + скрины переписки / видео.
10. **FAQ возражений** «нет слуха / стесняюсь / это эзотерика».
11. **Юр-сигналы** — лицензия, реквизиты, политика (нужно для модерации Яндекса).
12. **A/B оффера:** «бесплатная диагностика» vs «бесплатный голосовой аудит».

## Стратегические заметки
- Тёмный дизайн = отстройка, но для холодной женской VK-аудитории добавить тепла (светлые зоны, живые лица, дружелюбный тон) — снижает «агентский холод».
- Цену не показывать (как Этери/ЛАБвойс) — подтверждается ценовым ресёрчем. Программа-ориентир 45–70k ₽.
- Лучший быстрый канал на старте — **VK-таргет с видео реальных занятий** (лид 170–520 ₽), параллельно Директ на горячие запросы.
