# Brand Direction — Прояви

## Концепция

**Прояви** — пространство для проявления. Не курс. Не школа. Практика свободного голоса как канал к себе.

Голос — самый прямой и самый честный канал, через который мы существуем среди людей. Когда он свободен — свободен ты. Когда зажат — ты тоже. Поэтому мы работаем не только с вокалом, но с тем, что мешает звучать: с телом, с психикой, с привычкой молчать.

## Типографика

### Шрифты

- **Заголовки**: **Cormorant Garamond** (Google Fonts) — serif с характером, премиум-творческое ощущение. Веса 300, 400, 500, 600, 700.
- **Тело**: **Inter** (Google Fonts) — нейтральный технологичный sans, отличная читабельность. Веса 300, 400, 500, 600, 700.
- **Display акценты** (опционально, для отдельных слов): **Fraunces** — variable serif с возможностью реактивности на скролл. Веса 300–900, opsz 9–144.

### Принципы

- Заголовки в Cormorant, веса 400-500 (не bold — теряется характер serif)
- Тело в Inter weight 400, line-height 1.6
- `text-wrap: balance` для заголовков (нативно балансирует строки)
- `text-wrap: pretty` для параграфов (избегает «висячих» слов)
- Letter-spacing для крупных заголовков: -0.02em (немного теснее, более премиально)

## Tone of voice — 3 принципа

1. **На «ты», но без панибратства.** Тон близкого друга, который сам через это прошёл. Не «давайте раскроемся вместе!», а «попробуй услышать себя».
2. **Глубоко, но коротко.** Одно предложение должно бить точно. Длинные абзацы только когда правда нужно объяснить.
3. **Поэзия там, где техника не работает.** Тема голоса позволяет образность. Где нет сухой логики — можно метафорой.

### Не делаем

- ❌ Восклицательных знаков (одного на странице хватает)
- ❌ «Прокачать», «гарантировать результат», «изменим вашу жизнь»
- ❌ Эмодзи в тексте (только в CTA если совсем уместно)
- ❌ Капс везде, кроме одного-двух стратегических мест
- ❌ Корпоративных «мы команда профессионалов» — мы конкретные люди

### Делаем

- ✅ Образ через метафору («звучи как ты есть»)
- ✅ Короткие фразы рядом с длинными — ритм
- ✅ Прямые вопросы вместо утверждений («Узнаёшь себя?»)
- ✅ Конкретику вместо обобщений («15 минут с куратором» вместо «бесплатная консультация»)

## Easing

- `--ease-out`: cubic-bezier(0.22, 1, 0.36, 1) — стандартный «успокоенный» выход
- `--ease-in-out`: cubic-bezier(0.65, 0, 0.35, 1) — для длинных переходов
- `--ease-spring`: cubic-bezier(0.34, 1.56, 0.64, 1) — лёгкий пружинный (использовать осознанно и редко)

## Motion-принципы

По принципам Emil Kowalski / Jakub Krehel / Jhey Tompkins:

1. **300–400ms — стандартная длительность для интерактивных элементов** (hover, click).
2. **600–800ms — для крупных reveal'ов** (заголовки появляются).
3. **`prefers-reduced-motion`** обязательно учитывается — все нелинейные анимации отключаются.
4. **Никаких bouncy easing везде подряд** — overshoot только осознанно, для редких моментов.
5. **GPU-accelerated only** — анимируем `transform` и `opacity`, не `top/left/width/height`.
6. **Stagger для группы элементов** — если появляется 6 карточек, они появляются с задержкой 80-100ms между каждой.
