# HANDOFF — проект «Прояви» (лендинг)
_Обновлено: 2026-05-29. Для продолжения в новой сессии Claude Code._

## Что это
Обезличенный продающий лендинг для онлайн-школы вокала + раскрытия голоса/«проявленности».
Воронка: трафик → **бесплатная диагностика** (= ценная консультация по карте 6 компетенций) → менеджер созванивается → подбор маршрута → программа.
6 направлений: вокал · сценическая речь · работа перед камерой · психология проявленности · тело/дыхание · запись песни.
Сделан на базе анализа eteri.ru + переработанной диагностической карты (`Downloads\Диагностическая карта 2026.pdf`).

## Известные решения/ограничения
- **Цену на сайте НЕ показывать** (как eteri/ЛАБвойс). Ориентир по ресёрчу: программа 8 недель ~45–70k ₽. См. docs/competitor-research.md.
- **НЕ писать «40 минут»** про диагностику — пользователь явно просил убрать. Сейчас везде убрано — НЕ возвращать.
- before/after = авто-кроссфейд CSS (НЕ слайдер — пользователю не нравился drag).
- Higgsfield (banana MCP `mcp__3090960f-...`): баланс был ~350+ кр. Soul 2.0 ~0.12кр/фото, Kling видео ~10кр. Стиль ассетов: тёплый Kodak Portra, dark moody / экстатичные поющие лица.
- **Preview-инструмент (Claude_Preview) в прошлой сессии глючил:** скриншоты таймаутили на тяжёлой анимир. странице, вьюпорт схлопывался в 25px. Обход: `?screenshot=1` режим (паузит видео/анимации/грейн) в main.js; либо проверять через preview_eval. CSS кэшируется — для проверки правок нужен ?v=timestamp ИЛИ рестарт сервера.

## Что пользователь любит/не любит (важно!)
- НЕ любит опросники AskUserQuestion — принимать разумные решения сам.
- Любит: яркий-но-стильный дизайн, тёплые живые лица «в кайфе/экстазе», конкретику.
- НЕ любит: нумерацию разделов/кнопок (убрана), большие пустоты, «холодный агентский» вид, AI-«одинаковость» фото.
- Просил: продающий дизайн, много точек заявки, ясность для холодного трафика, тянуть по воронке.
- Хочет рекламу: VK-таргет (лид 170–520₽, формат — живое видео занятий) + Яндекс.Директ.
